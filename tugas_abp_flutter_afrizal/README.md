# 🚀 Fitur dan Implementasi Widget

Berdasarkan tampilan aplikasi, berikut beberapa widget Flutter yang diimplementasikan:

## 🧱 AppBar & Scaffold
Menggunakan `Scaffold` sebagai struktur utama aplikasi dan `AppBar` untuk menampilkan identitas mahasiswa pada bagian header aplikasi.

---

## 🎨 Stack & Container
Menggabungkan beberapa widget menggunakan `Stack` agar elemen dapat ditampilkan bertumpuk.

`Container` digunakan untuk:
- Memberikan warna latar belakang
- Mengatur padding dan margin
- Membuat tampilan lebih rapi dan responsif

---

## 📋 ListView
Menampilkan daftar item secara vertikal.

Implementasi meliputi:
- `ListView` manual untuk data statis
- Potensi penggunaan `ListView.builder` untuk data dinamis

---

## 🟦 GridView
Digunakan untuk menampilkan item dalam bentuk grid/kisi.

Cocok digunakan untuk:
- Menu aplikasi
- Galeri
- Tampilan item yang terstruktur rapi
